Application metéo avec une api dédier. 
Permet de rentré le nom d'une ville ou d'une commune pour
affiché un point avec toute les information et peut faire 
le tracé GPS si on ajoute une deuxième ville/commune.

Pour lancé l'application soit en la mettant sur un serveur
comme Xamp ou appache ou soit en lancent directement sur 
VS Code avec l'extention "Live Server".
